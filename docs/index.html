The Thesis

Most people assume AI-generated UI tops out at generic Material Design.

The difference is language.

Describing what using an interface feels like is almost always more powerful than describing what it looks like. When you compress real design intent into sensory vocabulary, you stop getting default output and start getting output with a point of view.

Not "animate card border" but "the chart exhales when you stop looking."
The vocabulary is the design tool.
